# mingw-dist

Contains build of mingw sourced from https://github.com/niXman/mingw-builds-binaries/releases

In the future may do the building itself as well.

## Note

This unfortunately has to be distributed as a .7z file which is not supported by Windows OOTB.

This is because using anything but lzma leads to 2xing the size and I don't want to waste 100mb of users network bandwidth.

In lde, this will be handled by setting up 7z locally and using it.
